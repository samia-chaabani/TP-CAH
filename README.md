# 🌳 TP — Classification Ascendante Hiérarchique (CAH)

## 📌 Description
Travaux pratiques sur la Classification Ascendante Hiérarchique (CAH)
réalisés à ENICarthage.

## 🎯 Objectifs
- Comprendre le principe du clustering hiérarchique
- Construire et interpréter un dendrogramme
- Choisir le nombre optimal de clusters
- Comparer avec K-means

## 📁 Contenu
- Notebooks Jupyter avec les implémentations complètes
- Datasets utilisés pour l'analyse

## 🛠️ Technologies utilisées
- Python
- Pandas
- Scikit-learn (AgglomerativeClustering)
- Scipy (dendrogram, linkage)
- Matplotlib
- Seaborn
